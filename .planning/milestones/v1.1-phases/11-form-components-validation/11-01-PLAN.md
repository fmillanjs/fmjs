<objective>
Install the two missing Shadcn UI components required for accessible form migration: Form (provides ARIA context wiring) and Select (replaces native select elements).

Purpose: All 12 form files in the project are missing aria-invalid and aria-describedby attributes. The Shadcn Form component is the ARIA automation layer that fixes this without manual coding — it generates IDs via React.useId(), sets aria-invalid on FormControl when errors exist, and wires aria-describedby to link controls to their error messages. The Select component replaces native select elements which have poor screen reader support.

Output: apps/web/components/ui/form.tsx and apps/web/components/ui/select.tsx
</objective>
