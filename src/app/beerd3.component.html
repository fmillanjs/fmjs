Craft Beer ABV and IBU correlation Description: D3 Graph to find the correlation between ABV (Alcohol by volume) and IBU (International Bitterness Units) if there is any, from a sample data of 40 different types of craft beers. Process:

  • Grabbed data from Kaggle datasets
  • Created a scatter graph with its axis
  • Added data using D3
  • Added interaction of brush zoom
  • Double-Click to zoom out.
